# 🔧 Cron 表达式修复说明

## ⚠️ 问题

GitHub Actions 的 cron 表达式格式:

```
❌ 错误: '0 */30 * * * *'  (6位 - GitHub 不支持)
✅ 正确: '*/30 * * * *'    (5位 - GitHub 标准)
```

## ✅ 已修复

**GitHub 上的文件已正确**: `.github/workflows/fetch-hot.yml`

```yaml
on:
  schedule:
    - cron: '*/30 * * * *'  # ✅ 每30分钟
```

## 📋 常用 Cron 表达式

| 频率 | 表达式 | 说明 |
|------|--------|------|
| 每30分钟 | `*/30 * * * *` | 每小时0分和30分 |
| 每小时 | `0 * * * *` | 每小时整点 |
| 每天2点 | `0 2 * * *` | 每天凌晨2点 |
| 每周一9点 | `0 9 * * 1` | 周一早上9点 |

## 🎯 5位格式说明

```
* * * * *
│ │ │ │ │
│ │ │ │ └── 星期 (0-7, 0=周日)
│ │ │ └─── 月份 (1-12)
│ │ └───── 日期 (1-31)
│ └─────── 小时 (0-23)
└───────── 分钟 (0-59)
```
